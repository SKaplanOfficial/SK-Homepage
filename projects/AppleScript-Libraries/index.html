AppleScript Libraries

I've created a few AppleScript libraries to aide in development of automation scripts.

ImageLib

An AppleScript library for loading and modifying images using high-level syntax.

PDFLib

An AppleScript library for getting information about PDFs, extracting text, rotating or flipping pages, and more.

WebLib

An AppleScript library for getting properties of websites, extracting their visible text, saving them to PDFs or images, and more.
